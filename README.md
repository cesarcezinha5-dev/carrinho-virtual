# Carrinho Virtual Inteligente

Sistema desenvolvido em Python para organizar compras online.

## Funcionalidades

- Cadastro de produtos por link
- Captura automática de título
- Captura automática de preço
- Cadastro manual de preço quando necessário
- Categorias dinâmicas
- Listagem de produtos por categoria
- Total por categoria
- Remoção de produtos
- Relatório TXT

## Tecnologias

- Python
- Requests
- BeautifulSoup
- Regex
- JSON
